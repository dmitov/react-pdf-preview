## How to install

```sh
pnpm install @react-pdf/preview
npm install @react-pdf/preview
yarn instal @react-pdf/preview
```

## How it works
```package.json
"scripts": {
  "dev": "react-pdf --dir=src/templates --port=3000"
}
```
